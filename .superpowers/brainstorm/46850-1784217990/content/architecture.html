一次任务的确定性数据流

1. Snapshot固定 HEAD、working tree 指纹、目标文件和 diff。
2. Instructions按文件解析根规则、目录规则、同名文件规则。
3. Context构建最小相关上下文，记录每段来源与 token 预算。
4. Reviewers按用户勾选并行运行；每个 Agent 独立 prompt 和能力集。
5. Validate结构、行号、证据、重复项和本次变更归因过滤。
6. Synthesize主 Agent 汇总，不得凭空新增无证据 finding。
7. Optional Fix隔离 worktree 修复；门禁通过后人工或自动应用。
