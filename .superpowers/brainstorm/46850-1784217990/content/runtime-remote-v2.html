无鉴权不等于取消协议安全

Host / Origin 校验抵御 DNS rebinding 和跨站请求；CORS 默认关闭，API 只接受 JSON content type。
Secret 留在服务端API key、MCP token 不通过前端 API 返回，不写任务事件或浏览器日志。
沙箱策略不放宽绑定 0.0.0.0 不会自动开放容器网络、宿主目录或命令权限。
服务器路径语义仓库选择器展示和校验的是服务端路径，不尝试访问远程浏览器的本地磁盘。
审计仍保留记录来源 IP、User-Agent、操作、任务和 apply 结果，但不把它当成身份。
反向代理可选后续可由 Nginx/Caddy 提供 TLS 或鉴权；应用首版不强制依赖。
明确接受的风险：在 `0.0.0.0 + auth none` 下，任何能访问服务端口的人都可以发起 review、消耗模型配额、查看结果，并在启用 Fix/自动审批时修改服务器仓库。首版允许这样部署，但不会把它描述为安全的多用户部署。
