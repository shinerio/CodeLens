四个必须显式处理的本地安全风险

本地不等于不出机发送给 OpenAI/远程 MCP 的文件会离开本机；任务创建页显示数据目的地与排除路径。
浏览器攻击 localhost只绑定 127.0.0.1，校验 Host/Origin，并使用启动时随机 token；不开放宽泛 CORS。
Trace 泄露代码默认关闭 sensitive trace data；API key/MCP token 存系统钥匙串，不写 SQLite、日志或 RunContext。
恶意仓库执行容器默认无网无宿主凭据；未信任仓库禁止运行 install/test/build 和仓库内 Skill/MCP。
